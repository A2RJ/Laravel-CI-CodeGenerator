# MVCGenerator
 Generate MVC or Restfull API in Laravel, Codeigniter and Express
 - Looping 3x untuk create MVC (model dan controller) agar code lebih singkat
 - Saat berhasil create MVC, generate dan console.log di terminal untuk routes nya
 - Cek jika sudah ada maka tampilkan notif dan skip ke db selanjutnya
